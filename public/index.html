Mr.Holmes CRM

Este domínio na Vercel hospeda apenas páginas estáticas. O CRM completo é uma aplicação Streamlit (Python) e precisa rodar em plataforma com servidor contínuo ou Docker.

Por que apareceu 404?

A Vercel procura arquivos estáticos (index.html) ou funções serverless. O repositório tinha só código Python (crm_app.py), sem pasta de saída — por isso o deploy concluía em ~2 segundos e a URL retornava NOT_FOUND.

Como colocar o CRM no ar

  • Streamlit Community Cloud (recomendado, grátis): conecte o repo GitHub, app crm_app.py, use requirements.txt na raiz.
  • Render / Railway / Fly.io: use o Dockerfile do projeto (porta 8512).
  • VPS + Docker Compose: fluxo descrito em DEPLOYMENT.md e no GitHub Actions do repo.
