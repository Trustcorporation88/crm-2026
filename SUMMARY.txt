🚨 PROBLEMAS COMUNS (Resolvidos em 30 segundos)
═══════════════════════════════════════════════════════════════════════════════

❌ "Port already in use"
   → python troubleshoot.py
   → Vai sugerir: lsof -ti:8512 | xargs kill -9

❌ "Docker daemon not running"
   → Abrir Docker Desktop (Windows/Mac) ou `sudo systemctl start docker` (Linux)

❌ "PostgreSQL connection failed"
   → docker-compose logs postgres
   → Aguardar "ready to accept connections"

❌ "API not responding"
   → make logs-api
   → docker-compose restart api

❌ Qualquer outra coisa:
   → python troubleshoot.py
   → Diagnóstico automático com soluções específicas
